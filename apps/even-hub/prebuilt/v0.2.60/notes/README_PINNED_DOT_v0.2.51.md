# Send to G2 v0.2.51 - Pinned dot marker

Small UI refinement built on v0.2.50.

Changes:

- Pinned inbox items on the G2 display now use a small bullet/dot marker instead of the text `PIN`.
- Reader metadata also uses a dot marker for pinned items.
- Trying to delete a pinned item from the G2 menu now shows a short temporary glasses message: `Pinned. Unpin in app to delete.`
- Pin/unpin remains available only from the mobile WebView.
- Pinned items remain protected from deletion until unpinned in the mobile WebView.

Gesture routing from v0.2.49/v0.2.50 is unchanged.
