# Send to G2 v0.2.52 - Pinned star marker

Small G2 UI refinement built on v0.2.51.

## Changes

- Pinned items are marked with a star (`★`) on the G2 menu instead of a dot.
- The G2 reader metadata/help marker also uses `★` for pinned items.
- Pin/unpin remains controlled only from the mobile WebView.
- Pinned items remain protected against deletion until unpinned in the mobile WebView.
- The short delete-blocked message remains: `Pinned. Unpin in app to delete.`

No gesture-routing changes.
